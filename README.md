## ✨ O QUE É O PROJETO?

</div>

O **IFCE Planner** é um ecossistema acadêmico completo feito para facilitar a vida do estudante. Une uma estética delicada e "fofa" com ferramentas poderosas de cálculo, organização e gestão de tempo.

---

<div align="center">

## 🎀 FUNCIONALIDADES PRINCIPAIS

</div>

* **📅 CRONOGRAMA INTELIGENTE:** Visualização dinâmica de aulas com destaque visual para o dia atual e integração de horários de ônibus.
* **✅ CHECKLIST NO CARD:** Sistema de tarefas diretamente na home com adição rápida, suporte a datas e limpeza de itens concluídos.
* **📊 DIÁRIO DE NOTAS:** Gestão de desempenho com cálculo de médias ponderadas e simulador de "quanto falta" para atingir a meta 7.0.
* **📂 BACKUP UNIFICADO:** Exportação e importação de todos os dados (configurações e notas) em um único arquivo JSON.

## 🚀 COMO INSTALAR

</div>

1.  Faça o **Download** dos arquivos `index.html` e `notas.html`.
2.  Mantenha ambos na **mesma pasta** para que os links funcionem corretamente.
3.  Abra o `index.html` em qualquer navegador (Chrome, Edge, Safari).

> 🎀 **Dica Mobile:** No telemóvel, use a opção "Adicionar ao ecrã principal" no seu navegador para transformar o planner em um App!
