## ✨ O QUE É O PROJETO?

O **IFCE Planner** é um ecossistema acadêmico completo feito para facilitar a vida do estudante. 
Une uma estética delicada com ferramentas poderosas de cálculo e organização.

---

## 🎀 FUNCIONALIDADES PRINCIPAIS

**📅 CRONOGRAMA INTELIGENTE** Visualização dinâmica de aulas com destaque para o dia atual e horários de autocarro integrados.

**✅ CHECKLIST NO CARD** Sistema de tarefas com adição rápida, datas e remoção de itens concluídos.

**📊 DIÁRIO DE NOTAS** Cálculo de médias ponderadas e simulador de nota faltante (Meta 7.0).

**📂 BACKUP UNIFICADO** Exportação de todos os dados e notas em um único arquivo JSON para nunca perder nada.

## 🚀 COMO INSTALAR

1. Faz o **Download** dos arquivos `.html`.
2. Mantém o `index.html` e o `notas.html` na **mesma pasta**.
3. Abre o `index.html` no teu navegador.

*Dica: No telemóvel, clica em "Adicionar ao ecrã principal" para usar como App!*
